# 🏎️ McLaren Landing Page

A passion project built for fun — because why settle for a boring portfolio when you can obsess over the two most extreme McLarens ever made?

**→ [Live Site: snowy-monsoon-jvwb.here.now](https://snowy-monsoon-jvwb.here.now/)**

---

## What is this?

This is a landing page I built to showcase my two all-time favourite cars: the **McLaren P1** and the **McLaren 600LT**. One is a hybrid hypercar and the other is a lightweight track weapon that makes every other sports car feel numb. Both are absolutely mental, and I love them.

Built entirely for fun with the help of **Claude** (Anthropic's AI) — I described what I wanted, Claude wrote it, we argued about TypeScript types, and somehow ended up with this.

---

## The Cars

### McLaren P1 — The Hypercar

- 903 hp hybrid powertrain (V8 + electric motor)
- 0–60 mph in 2.8 seconds
- Active aerodynamics that generate 600 kg of downforce
- One of only 375 ever made
- Basically a road-legal F1 car

### McLaren 600LT — The Driver's Car

- 592 hp twin-turbo V8
- Lightest car in its class (1,247 kg)
- Top-exit exhaust that sounds like a jet engine
- Named after the legendary McLaren F1 GTR "Longtail"
- Pure, unfiltered driving experience

## Built With

- **React + TypeScript** — component structure
- **Vite** — blazing fast build tool
- **Tailwind CSS** — styling
- **Framer Motion** — animations and parallax effects
- **Lucide React** — icons
- **Claude (Anthropic)** — my AI co-pilot for the whole thing
- **here.now** — hosting

## Features

- 🌑 Full dark mode with carbon fibre textures
- ✨ Animated particle field background
- 🖱️ Mouse-tracked parallax on car images

- 📱 Fully responsive
- ⚡ Custom SVG car illustrations

## Run It Locally

```bash
git clone <this-repo>
cd mclaren-landing
npm install
npm run dev
```

To deploy:

```bash
npm run deploy
```
